# Loan Predictor Model

This project uses machine learning to predict whether a loan should be approved based on applicant details like income, credit history, loan amount, and employment status.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## Features
- Data preprocessing
- Model training
- Loan prediction
- Model evaluation

## How to Run
1. Clone the repo
git clone https://github.com/yourusername/loan-predictor-model.git

2. Install requirements
pip install -r requirements.txt

3. Run the project
python predictor.py

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
